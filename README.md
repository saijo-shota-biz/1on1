## 📘 このLPについて

エンジニアとしてのキャリアや、自分自身の思考に向き合いたい若手エンジニアのための対話セッションを紹介しています。

- 技術指導ではなく、**思考の整理や問いの言語化**を目的とした時間
- 自分自身のコーチング・ファシリテーション力を磨く“実践の場”としての取り組み
- **最低500円〜の投げ銭形式（Pay What You Want）**

1on1の雰囲気や目的、セッションの進め方を丁寧に伝えるLPとして設計しました。
